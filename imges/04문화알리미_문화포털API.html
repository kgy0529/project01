  • 문화포털
    오픈 API 사용안내

    기본적으로 제공 API 서비스를 사용하시기 위하여 ‘서비스 인증키’ 를 개별적으로 발급받으셔야 합니다. 발급기관은 아래와 같습니다.

    공공데이터포털 http://www.data.go.kr
  • 오픈API
    <URL> 조합 예시

    요청URL + 검색조건 값(파라미터) + Service Key (Service Key에 대한 부분은 UTF-8로 URL 인코딩하여 보내야 함)

    http://www.culture.go.kr/openapi/rest/publicperformancedisplays/period?from=20101118&to=20101217&cPage=1&rows=10&place=&gpsxfrom=&gpsyfrom=&gpsxto=&gpsyto=&keyword=&sortStdr=1&serviceKey={서비스 키}

    ※ 자세한 사항은 각 조회서비스별 개발가이드를 참고하시기 바랍니다.
